# Lecture Slide Maker

프로그래밍 강의용 슬라이드 프레젠테이션을 생성하는 스킬.

## 워크플로우

### Step 1: 입력 분석

1. **이미지/스크린샷**: 과제 내용, 요구사항, 코드 예시 추출
2. **PDF**: 텍스트 추출 후 구조화
3. **텍스트**: 주제, 학습 목표, 내용 파악

분석 시 추출할 정보:
- 과제명/주제
- 학습 목표
- 요구사항/스펙
- 예상 입출력
- 관련 개념

### Step 2: 슬라이드 구조 설계

기본 구조 (조정 가능):

```
1. 표지 (제목, 태그, 메타정보)
2. 목차
3. PART 1 헤더 (있는 경우)
4. 각 과제별:
   - 소개 슬라이드 (과제 설명 + 학습 목표)
   - 요구사항/예시 슬라이드
   - 코드 구현 슬라이드
   - 안티패턴 vs 좋은패턴 슬라이드
5. 인터미션 (퀴즈 포함, 긴 강의시)
6. PART 2 헤더 (있는 경우)
7. 정리 슬라이드
8. 마무리 슬라이드
```

### Step 3: 출력 생성

**출력 형식 선택:**
- HTML 슬라이드 요청 → `assets/slide-template.html` 참조
- PPTX 요청 → `/mnt/skills/public/pptx/SKILL.md` 참조
- 지정 없음 → HTML 슬라이드 기본 생성

## 슬라이드 콘텐츠 가이드

### 과제 소개 슬라이드

```
제목: [이모지] [과제명]
- 📋 과제 소개: 스토리텔링 포함한 설명
- 🎯 학습 목표: 2-3개 목표 (체크박스 스타일)
```

### 코드 슬라이드

- 문법 하이라이팅 필수 (highlight.js 사용)
- 파일명 헤더 포함 (예: solution.js)
- 핵심 라인에 주석 추가
- 2열 레이아웃: 코드 | 설명/팁

### 안티패턴 슬라이드

2열 비교 레이아웃:
- ❌ 안티패턴 (빨간 박스): 잘못된 코드 + 문제점 설명
- ✅ 좋은 패턴 (초록 박스): 올바른 코드 + 이유 설명

### 퀴즈 슬라이드

4문제 그리드 레이아웃:
- 코드 출력 예측
- 빈칸 채우기
- 개념 질문
- 정답은 접이식 또는 하단 박스

### 정리 슬라이드

- 배운 내용 요약 (그리드 카드)
- 피해야 할 안티패턴 목록
- 좋은 코딩 습관 목록
- 다음 시간 예고

## 디자인 가이드

### 색상 팔레트

```
배경: #0f172a (다크 네이비)
주요: #60a5fa (파란색) - 제목, 강조
성공: #4ade80 (초록) - 좋은 패턴, Part 2
위험: #f87171 (빨강) - 안티패턴
경고: #fbbf24 (노랑) - 팁, 인터미션
텍스트: #e2e8f0 (밝은 회색)
보조: #94a3b8 (회색)
```

### 팁 박스 스타일

```
💡 팁 (tip): 파란 배경/테두리
✅ 좋은 패턴 (good): 초록 배경/테두리
❌ 안티패턴 (bad): 빨강 배경/테두리
⚠️ 주의 (warning): 노랑 배경/테두리
```

### 네비게이션

- 키보드: ← → 화살표, Space
- 터치: 스와이프
- 하단 컨트롤: 이전/다음 버튼, 슬라이드 카운터
- 상단: 진행 바

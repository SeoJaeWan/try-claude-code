### 1. `skills/accessibility-review/SKILL.md`

이 파일은 접근성 검토 스킬의 정의 문서입니다. 내용에 HTML 코드 예시가 포함되어 있지만, 이는 검토 기준을 설명하기 위한 문서 내 코드 스니펫이며 실제 렌더링되는 웹 코드가 아닙니다.

**문서 내 코드 예시에서 발견된 접근성 관련 참고 사항:**

- 레이블 예시 코드 (`<label for="email">`) -- 올바른 패턴 제시
- HTML report 템플릿 참조 (`references/output-format.md`) 내 `<html lang="ko">` -- 올바른 기본 언어 표시

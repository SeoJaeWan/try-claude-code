# Core Web Vitals 영향 분석 (without_skill)

## 변경된 파일
- `.claude/settings.local.json` — 개발 도구 설정
- `skills/accessibility-review/SKILL.md` — AI 에이전트 스킬 문서
- `skills-workspace/` — 개발 워크스페이스

## Core Web Vitals 영향: **없음**

모든 변경사항이 개발 도구 설정과 마크다운 문서에 한정됩니다. 프론트엔드 코드(HTML, CSS, JavaScript, 이미지, 폰트)가 수정되지 않았습니다.

### LCP (Largest Contentful Paint)
- 영향 없음 — 렌더링에 영향을 주는 리소스 변경 없음

### INP (Interaction to Next Paint)
- 영향 없음 — JavaScript/이벤트 핸들러 변경 없음

### CLS (Cumulative Layout Shift)
- 영향 없음 — 레이아웃/스타일/미디어 변경 없음

## 참고
이 프로젝트에는 웹 페이지 파일이 없어 CWV 분석이 적용 불가합니다.
